◆ receiveInfraredCommand()

uint16_t lib::receiveInfraredCommand ( )

Use the infrared receiver to receive data from the transmitter robot, or fall back to the button in case of failure (address will be 0 in that case). Blocks until data is received, either through the infrared photodetector or the interrupt button Note: in debug mode, requires initializeUsart() to have been called beforehand

Returns
The digit number sent through the SIRC protocol or the number of times the button was pressed instead
